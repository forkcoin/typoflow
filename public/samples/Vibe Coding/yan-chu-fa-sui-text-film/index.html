Opening / 00

你说了很多话但结果没有动

这个时代重新要求我们学会表达，不是因为话变少了，而是因为话和结果之间的距离，被看得更清楚了。

Scene / 01

第一处失真，通常藏在一个小词里。

“帮我买瓶大一点的牛奶。”

说话者心里2L
执行者理解1L
问题来源大一点

不是对方不认真，而是指令没有给出可以落地的尺寸。

Scene / 02

到了协作现场，模糊词会变成返工。

尽快完成市场调研报告 数据要详实 分析要深入 下周一例会前给我
尽快 详实 深入 高质量

这些词听起来专业，却没有告诉人：时间、范围、标准和交付物到底是什么。

Scene / 03

AI把表达里的问题放大了。

画一个美丽的日落 写一篇关于环保的深度文章
输入美丽、深度、好看
机器理解训练数据里的平均答案
输出像对，又不真正对

Scene / 04

过去还能糊弄过去，现在不行了。

信息不再稀缺，注意力才稀缺。
远程协作拿走了表情、语气和眼神。
AI不会替你脑补，只会执行你给出的字面信息。

表达不清，不只是慢一点，而是会失去被理解、被执行、被相信的机会。

Scene / 05

表达不是信息的包装，表达就是行动的起点。

“把房间收拾干净。”

孩子理解玩具堆到角落
你的期待物品归位、床铺整洁
真正缺失标准

Scene / 06

这不是提示词教材，而是表达训练。

咒语 模板 套话 目标 背景 边界 标准

提示词只是一个应用场景。真正决定结果的，是你能不能把想法说成别人和 AI 都能执行的结构。

Scene / 07

这本书的学习路径，是一条校准链。

觉察：发现表达在失真。
构建：先说清目标、背景和边界。
校验：说清标准，拆解任务。
迭代：用反馈和多轮校准逼近结果。
迁移：把能力用到人、团队和 AI 场景。

Ending / 08

你的语言，不只是声音或文字。

当目标清楚、背景充分、边界明确、标准可查，表达才会从“说过了”变成“做成了”。
